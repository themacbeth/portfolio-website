THE RITUAL

A 5-minute generative audiovisual ceremony. Sacred geometry meets circuit-board architecture — concentric ring systems, radial spokes, right-angle traces connecting nodes, and futuristic script inscribed between mandala layers. Every element is alive, displaced by Perlin noise, breathing with the drone.

The experience progresses through five phases — Genesis, Inscription, Procession, Rapture, Still — building from a single point to maximum complexity, then deconstructing back to silence. Three arcade buttons allow override, but the piece is self-sufficient. It witnesses itself.

live · interactive

p5.js · Web Audio API · Bayer Dithering · 2026

Chrome Maximals. Circuit-board faces. Holographic vestments. Belief and machinery have already fused.

MARFA INSTALLATION

Designed for a gallery space in Marfa, Texas — a 2m × 2m LED wall with a laser projector firing over the audience. Three jumbo illuminated dome buttons on a pedestal in the center of the room. The constraint of three buttons makes it feel like a public instrument. Each press is a small act of worship in a techno-religious order.

ART BLOCKS — 360 SACRED SCRIPTS

Each mint is a unique, hash-deterministic sacred script. The same hash always produces the same mandala — same symmetry order, same ring configuration, same circuit traces, same inscriptions. 360 pieces. Pure JavaScript, no dependencies, on-chain ready.

Traits are drawn from the lore — Station, Inscription, Rite, Vestment, Temperament, Witness, Halo, Veil. The code is the sacred text. ~12% of mints contain a hidden inscription that reveals after sustained viewing.
